## Pokédex Dashboard

A responsive web application that displays data for the first 150 Pokémon. Users can view Pokémon details, search by name, filter by type, and toggle between dark and light mode. This project is built using HTML, CSS, and JavaScript and consumes the public PokéAPI.

## Features

Search: Dynamically search for any Pokémon by name.
Filter: Filter the homepage grid by Pokémon type (supports dual types).
Dark mode: Toggle between light and dark mode (powered by CSS variables).
Interactive Cards: Hover animations, flip card feature, Pokémon cry sound on click, and animated sprites.
Error Handling: Loading spinner and API error display.
Responsive Design: Fully responsive across mobile, tablet, and desktop.

## Design

Design elements and animations were implemented using HTML5, CSS3, and JavaScript.
The app uses CSS 3D transforms for card flipping and CSS variables for theme switching.
